Sample Exam 2

Part A: Conceptual (Multiple Choice)

1. What does a small p-value indicate?

  1. The null hypothesis is true
  2. Strong evidence against the null hypothesis
  3. The sample size is small
  4. The model has no error

2. What does R² measure?

  1. The slope of the regression line
  2. The variability in the predictor
  3. The proportion of variability explained by the model
  4. The p-value of the test

3. A Type I error occurs when:

  1. You fail to reject a false null hypothesis
  2. You reject a true null hypothesis
  3. You accept a false alternative hypothesis
  4. The sample size is too large

4. Which test is used to compare more than two group means?

  1. t-test
  2. ANOVA
  3. Chi-square
  4. Regression

5. In regression, what does b₁ represent?

  1. The average of Y
  2. The predicted value when X = 0
  3. The average change in Y for a one-unit increase in X
  4. The variance of Y
